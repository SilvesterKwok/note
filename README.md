### 提交方式：
  1. git bush提交
    1. git add . （注：别忘记后面的.，此操作是把Test文件夹下面的文件都添加进来）
    2. git commit  -m  "提交信息"  （注：“提交信息”里面换成你需要，如“first commit”）
    3. git push -u origin master   （注：此操作目的是把本地仓库push到github上面，此步骤需要你输入帐号和密码）
  2. atom提交
